# Website HIMASI IT Mojosari

Website resmi Himpunan Mahasiswa Sistem Informasi Institut Teknologi Mojosari.

## Stack
- HTML5 & CSS3
- PHP
- Bootstrap 5
- Vanilla JavaScript
- AOS (Animate On Scroll)

## Halaman
- **Home** — Hero, statistik, divisi, berita terbaru
- **Tentang Kami** — Sejarah & visi misi organisasi
- **Struktur Organisasi** — Bagan pohon kepengurusan interaktif
- **Program Kerja** — Filter divisi instan (tanpa reload)
- **Berita** — Pencarian & filter kategori realtime
- **Kontak** — Form AJAX, info sekretariat, peta kampus

php -S localhost:8000
