# Website HIMASI IT Mojosari

Website resmi Himpunan Mahasiswa Sistem Informasi Institut Teknologi Mojosari.

## Stack
- HTML5 & CSS3
- PHP
- Bootstrap 5
- Vanilla JavaScript

## Cara Menjalankan

Jalankan perintah ini di terminal:

```bash
php -S localhost:8000
```

Lalu buka [http://localhost:8000](http://localhost:8000) di browser.
